1. Explain CSS Box Model

The CSS box model defines how an element is sized and spaced.
It consists of content, padding, border, and margin.
Padding is inside the border, margin is outside.

2. Difference between class and id selectors

A class selector is used for styling multiple elements, while an id selector is used for a single unique element.
IDs have higher priority than classes.

4. Describe Flexbox layout

Flexbox is a layout system used to arrange items in one direction (row or column).
It helps in easy alignment, spacing, and centering.

5. Explain CSS Grid basics

CSS Grid is a layout system that works in two directions (rows and columns).
It is used to create complex page layouts.

6. What is responsive web design?

Responsive web design makes a website adjust automatically to different screen sizes and devices.
